# AI-Powered Library Management System

## Project Overview

The AI-Powered Library Management System is a web-based application developed using Flask and SQLite to automate and simplify library operations. The system enables librarians and students to manage books efficiently through secure login, book issue and return management, inventory tracking, and intelligent book recommendations based on user borrowing history.

---

## Objective

The primary objective of this project is to automate library management activities, reduce manual effort, improve book inventory control, and provide personalized book recommendations to users. The system also helps librarians monitor borrowing activities, manage records efficiently, and analyze reading trends for better decision-making.

## Installation Steps

### Step 1: Clone Repository

```bash
git clone https://github.com/DarshilAgarwal19/library-management-ai.git
```

### Step 2: Open Project Folder

```bash
cd library-management-ai
```

### Step 3: Install Dependencies

```bash
pip install Flask Werkzeug
```

### Step 4: Run Application

```bash
python app.py
```

### Step 5: Open Browser

```text
http://127.0.0.1:5000
```
